***Pizza Sales SQL Analysis***

A complete SQL-based data analysis project exploring revenue trends, ordering patterns, product performance, and advanced analytics using the Pizza Sales dataset.

***Project Overview***

This project analyzes a Pizza Sales dataset using SQL to uncover key business insights that can support data-driven decision-making. The analysis includes basic, intermediate, and advanced SQL queries ranging from revenue calculations to advanced techniques like moving averages, forecasting, basket analysis, and A/B comparisons.

***Dataset Structure***

The dataset includes the following tables:

orders – order_id, order_date, order_time

order_details –  order_id, pizza_id, quantity

pizzas – pizza_id, size, price, pizza_type_id

pizza_types – pizza_type_id, name, category, ingredient

***Key Insights***

Large pizzas generate the highest revenue.

Friday and Saturday have peak order volumes.

Revenue shows clear weekly and monthly patterns.

Certain pizza types strongly pair together (basket analysis).

Moving averages help smooth weekly fluctuations for trend detection.
